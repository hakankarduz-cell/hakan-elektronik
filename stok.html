🔥 Firebase Bulut Senkronizasyonu
✅ Firebase bağlı — Tüm cihazlar senkronize
⚠️ Firebase config bilgilerini girin. Nasıl alırım? — Aşağıdaki adımları takip edin.
🔥 Firebase Kurulum (5 dakika): 1. console.firebase.google.com adresine gidin 2. "Proje oluştur" → proje adı yazın (örn: hakan-elektronik) → Devam edin 3. Google Analytics'i kapatıp "Proje oluştur" deyin 4. Sol menü → "Realtime Database" → "Veritabanı oluştur" → "Test modunda başlat" seçin 5. Proje Ayarları (⚙️) → "Genel" sekmesi → aşağı kaydırın → "Web uygulaması ekle" ( ikonu) 6. Uygulama adı girin → "Uygulamayı kaydet" 7. Açılan firebaseConfig nesnesindeki değerleri yukarıdaki alanlara kopyalayın ✅ Hepsi bu kadar! Tüm telefonlarda aynı config'i girin.
